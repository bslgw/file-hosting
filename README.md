# 安装

登录 cloudflare.com

新建 worker

复制粘贴 worker.js 中的全部代码

创建KV空间并绑定 变量名：FILE_STORE 

保存

# 说明

txt，m3u 类型的文档可以直接在线编辑，编辑时需要输入的密码，密码就是上传时设置的密码
